# Drawing Requirements - Synthetic

All dimensions are in millimetres.

| Feature | Minimum diameter | Maximum diameter |
|---|---:|---:|
| Hole H | 25.000 | 25.021 |
| Shaft S | 24.975 | 24.990 |

Requirements:

- Values are diameters.
- No coating or thermal allowance applies.
- A measured pair is acceptable only when each feature is within its own stated limits.
- Fit classification is based on the full limit range, not only the measured pair.
