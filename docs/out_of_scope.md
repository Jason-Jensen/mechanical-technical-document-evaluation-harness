# Explicitly Out of Scope for Month 1

Do not add:

- OpenClaw installation or agent execution;
- any other agent harness;
- model or vendor API calls;
- automated LLM grading;
- trajectory collection;
- process-reward modeling;
- reward-model training;
- reinforcement learning;
- task adaptation from agent failures;
- RAG, embeddings, or vector databases;
- FastAPI or other web APIs;
- SQL or persistent databases;
- PDF parsing, OCR, computer vision, or native CAD parsing;
- GUI automation;
- Docker, cloud hosting, or deployment;
- dashboards or frontend work;
- GitHub Actions or broader CI/CD;
- benchmark statistics beyond simple case counts;
- consulting offers or client delivery;
- engineering sign-off, code-compliance opinions, or safety-critical release decisions.

Also prohibited:

- confidential Outlier prompts, rubrics, screenshots, reviewer feedback, or internal workflows;
- employer or client files without explicit authorization;
- production credentials or personal accounts;
- copied proprietary drawings or close reconstructions that expose protected content.

Month 1 is complete when the case specifications are trustworthy, not when an agent can run them.
